# Erweiterung

Neue Module:
1. HTML-Datei in passendem Fach-/Bildungsgangordner anlegen.
2. MODULE-Konfiguration setzen.
3. Bei Bedarf neuen Generator in `assets/js/app-core.js` ergänzen.
4. Modul in `index.html` eintragen.

Regeln:
- Fach und Bildungsgang klar trennen.
- Keine personenbezogenen Daten speichern.
- Keine externen Bibliotheken verwenden.
- Aufgaben aus externen Quellen nicht 1:1 übernehmen.
