# Kaufmännisches Rechnen – Lernquest

Offline nutzbares Mathematik-Lernapp-System für BFS und Einzelhandel.

## Version
1.1.0

## Enthalten
- Dashboard mit XP, Level, Tagesmission und Badges
- Fortschrittsseite
- Badge-Seite
- Lernmodule zu Dreisatz, Prozentrechnung, Durchschnitt, Zinsen, Kasse und Kalkulation
- Textaufgaben in einfacher Sprache
- Musterlösung erst nach Eingabe und Prüfung
- Lehrkraft-Modus mit Code `150411`

## Datenschutz
Die App speichert nur lokale Lernstände im Browser. Es werden keine Namen, Klassenlisten, E-Mail-Adressen oder personenbezogenen Daten gespeichert.

## Offline-Nutzung
`index.html` im Browser öffnen. Keine externen Bibliotheken, keine CDNs, keine externen Schriftarten.
