1. Basisschema

Listeneinkaufspreis (LEP)100 %
- Lieferrabatt
= Zieleinkaufspreis (ZEP)
- Liefererskonto
= Bareinkaufspreis (BEP)
+ Bezugskosten
= Bezugspreis / Einstandspreis
+ Handlungskosten
= Selbstkostenpreis
+ Gewinn
= Barverkaufspreis
+ Kundenskonto
= Zielverkaufspreis
+ Kundenrabatt
= Listenverkaufspreis netto

2. Einkaufskalkulation

Vom Listeneinkaufspreis werden Nachlässe abgezogen. Bezugskosten werden addiert.

  • Lieferrabatt senkt den Einkaufspreis.
  • Liefererskonto senkt den Zahlbetrag.
  • Bezugskosten erhöhen den Bezugspreis.
LEP - Rabatt = ZEP · ZEP - Skonto = BEP · BEP + Bezugskosten = Bezugspreis

3. Zuschläge

Handlungskostenzuschlag
HKZ % = Handlungskosten · 100 / Bezugspreis
Gewinnzuschlag
Gewinn % = Gewinn · 100 / Selbstkostenpreis

4. Verkaufskalkulation im Hundert

Kundenskonto und Kundenrabatt werden häufig „im Hundert“ berechnet, weil der Nachlass vom später höheren Verkaufspreis abgezogen wird.

Zielverkaufspreis = Barverkaufspreis · 100 / (100 - Skonto %)
Listenverkaufspreis = Zielverkaufspreis · 100 / (100 - Rabatt %)

5. Kalkulationsmethoden

VorwärtskalkulationVom Einkaufspreis zum Verkaufspreis.
RückwärtskalkulationVom Marktpreis zurück zum maximalen Einkaufspreis.
DifferenzkalkulationEinkaufs- und Verkaufspreis sind gegeben. Gesucht ist der Gewinn.
